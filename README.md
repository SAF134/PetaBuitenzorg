# PetaBuitenzorg 🏨🏥🛍️⛽

Aplikasi Peta Interaktif Kota Bogor (Buitenzorg) berbasis web SPA (Single Page Application).

## Fitur Utama
- **Eksplorasi Lokasi**: Temukan Hotel, Rumah Sakit, Mall, dan SPBU di Bogor.
- **Peta Interaktif**: Visualisasi lokasi menggunakan Leaflet.js dengan fitur cluster.
- **Filter Canggih**: Filter berdasarkan rating, harga, fasilitas, dan jarak.
- **Persistensi Data**: Filter dan lokasi GPS tetap tersimpan saat navigasi halaman.
- **Desain Modern**: Antarmuka gelap (Dark Mode) yang responsif dan premium.

## Teknologi
- Vanilla JavaScript
- Leaflet.js (Maps)
- Lucide Icons
- Capacitor (Mobile Support)
- Vite (Build Tool)

## Cara Menjalankan
1. Clone repositori ini.
2. Jalankan `npm install`.
3. Jalankan `npm run dev`.
